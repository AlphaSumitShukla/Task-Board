This is a simple Task Board (Kanban) application built with HTML, CSS, and JavaScript. It allows users to manage tasks efficiently by visually organizing them into three stages: To Do, In Progress, and Done. Users can easily drag and drop tasks between columns to update their progress.

Key Features

Drag & Drop: Move tasks between columns effortlessly.

Responsive Design: Works well on desktop and mobile screens.

Interactive UI: Visual feedback when dragging tasks (hover effect and smooth transitions).

Task Management: Add, move, and delete tasks easily (optional feature for later upgrades).

Customizable: Simple codebase makes it easy to extend and style.

Tech Stack

HTML5 – Structure of the board and tasks

CSS3 – Styling, responsive design, and hover effects

JavaScript (ES6) – Drag & drop functionality and task interactions

Project Overview

This project is ideal for anyone who wants a lightweight, front-end task management tool without the need for a backend. It demonstrates the use of DOM manipulation, drag-and-drop events, and flexbox layout in creating interactive web applications.
